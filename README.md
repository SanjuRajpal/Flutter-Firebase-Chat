# Flutter-Firebase-Chat

- Configuration<br/>
  Step 1: Create app on firebase.<br/>
  Step 2: Download google-services.json for android & GoogleService-Info.plist for iOS.<br/>
  Step 3: Store files in respective path.<br/>
  Step 4: Run project.<br/>
